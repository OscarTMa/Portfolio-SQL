## ‚òÅÔ∏è Project Architecture
This project was deployed using a **Modern Data Stack** approach, moving away from local files to a fully cloud-based environment.

* **1. Ingestion:** Raw data (CSV) uploaded to **Google Cloud Platform (GCP)**.
* **2. Warehousing:** Data stored and processed in **Google BigQuery**.
* **3. Analysis:** SQL queries executed in the cloud using BigQuery Standard SQL.
* **4. Visualization:** **Tableau Public** connected via data extraction pipeline.

## üîé Key Insights
* **Content Dominance:** Historically, **Movies** represent about 70% of the catalog, though **TV Shows** have seen a significant increase in production since 2015.
* **Top Producers:** The United States, India, and the United Kingdom are the leading contributors to the platform's library.
* **Content Trends:** A sharp spike in content addition was observed between 2016 and 2019.

## üõ† SQL Skills Demonstrated
* **Cloud Computing:** Navigating GCP Console and managing BigQuery Datasets.
* **Data Cleaning:** Handling unstructured text data using `SPLIT`, `SAFE_CAST`, and `OFFSET` to extract numerical values from string columns (e.g., Duration).
* **Aggregation:** Using `GROUP BY` and `COUNT` to summarize large datasets.
* **Filtering:** Strategic use of `WHERE` clauses to handle `NULL` values.
